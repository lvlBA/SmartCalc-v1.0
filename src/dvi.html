SmartCalc_v1.0

About programm

The calculator is implemented on the QT graphic library. Also if x expression was used, function graph will be showed in a different window.

How to start

To start using the calculator, you need to install it using the make install command. This command will install the program on the desktop in the SmartCalc folder.

Uninstall

To unistall the programm you can do it using the make uninstall command.
